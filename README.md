# LoraWAN-RaspberryPi-ChirpStack-AzureIoTHub
The idea is to control a PLC over long distances (1km) using a LoRaWAN UC100 module via Modbus communication. The UC100 will then send information to the LoRaWAN gateway UG56. A Raspberry Pi will host the Chirpstack network server and handle MQTT communication with Azure IoT Hub.
